# exampleABMRS

`exampleABMRS` contains the R code used for numerical experiments in the article **Adaptive Bayesian Multivariate Spline Knot Inference with Prior Specifications on Model Complexity**.

The source package is available at: [ABMRS](https://github.com/junhuihe2000/ABMRS)

This repository includes implementations for knot inference, function fitting, and applications to real-world datasets.

## Overview

1. `fitting`: Evaluation of curve and surface fitting with jumping discontinuity.
2. `knot_inference`: Simulations of knot inference in linear spline models with one, two, and four knots.
3. `real_data`: Analysis of real datasets in univariate curve fitting (triceps skinfold thickness data).
4. `scripts`: Executable scripts for simulation and fitting workflows.
